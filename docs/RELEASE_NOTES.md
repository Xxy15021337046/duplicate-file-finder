## 🔄 升级指南

### 从v2.x升级到v3.0

1. **备份数据**: 备份现有的数据库文件（*.db）
2. **下载新版**: 下载v3.0.0安装包
3. **覆盖安装**: 解压到原目录（或新目录）
4. **保留数据**: 保留原有的数据库文件
5. **重启程序**: 重新启动程序即可

**注意**: 
- v3.0向后兼容v2.x的数据库
- 首次启动会自动迁移数据结构
- 建议先备份再升级

### 从v1.x升级到v3.0

建议全新安装：
1. 下载v3.0.0安装包
2. 解压到新目录
3. 手动导入v1.x的检测结果（如有需要）
